Endpoints

La sección de Registro de Endpoints es crucial para documentar las rutas y puntos finales de la API del proyecto, proporcionando una referencia central para desarrolladores, integradores y cualquier persona que interactúe con la interfaz del sistema.

Propiedad

Método Ruta del Endpoint Descripción
GET /api/homez/propiedad Obtiene la lista de propiedades registradas.
GET /api/homez/propiedad/{id} Obtiene una propiedad específica.
POST /api/homez/propiedad Crea una nueva propiedad.
PUT /api/homez/propiedad Actualiza una propiedad existente.
DELETE /api/homez/propiedad/{id} Borra una propiedad específica.
